- where

  1. column 조회 조건

     Select (column) from (table) where (con1) = (con2);

  2. and, or 조건

     Select (column) from (table) where (con1) = (con2) and(or) (con3)=(con4);

  3. like 조건

     Select (column) from (table) where (con1) % (con2);
